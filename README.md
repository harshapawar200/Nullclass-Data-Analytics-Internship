# Nullclass Data Analytics Internship Project

## Project Overview
This project was completed as part of my **Data Analytics Internship at Nullclass EdTech** .  
It includes six analytical tasks based on the same e-commerce dataset (`orders.csv`).

## Technologies Used
- Python (Pandas, NumPy, Matplotlib)
- Jupyter Notebook
- Data Visualization
- Exploratory Data Analysis

## Tasks Completed
1. Top 5 Products by Sales in 2022 (Mobiles & Tablets)
2. Identifying Customers Who Checked Out but Didn’t Pay in 2022
3. Comparing Weekend vs Weekday Sales in Q4 2022
4. Sales Decrease in “Others” Category (2021 vs 2022)
5. Finding Products with Largest Decrease (2021 vs 2022)
6. Comparing Category-wise Sales Trends in 2022

## Insights & KPIs
- Year-over-year sales changes  
- Customer purchase patterns  
- Weekend campaign effectiveness  
- Category performance comparison  

## Dataset
Dataset used: `orders.csv` (provided in training)
